# Claude Code Rules

## Plan files

- NEVER create plan files (*.md) in the project root directory.
- All plan documents must go in the `readme/` folder.
- Plan files must follow the naming convention: `YYYY-MM-plan-name.md` (e.g., `2026-03-cycle-dashboard.md`).
- If asked to create a plan, always place it at `readme/YYYY-MM-<descriptive-name>.md` using the current year and month.
- If you encounter an existing plan file in the root (like `plan.md`), suggest moving it to `readme/` with the proper naming convention.
